Money Manager Android App

  • REACT
  • REACT-NATIVE
  • CSS
  • JAVASCRIPT
  • EXPRESS
  • NODEJS
  • MONGODB

This was a MERN stack project I built for my girlfriend who wanted an app to easily manager her weekly spending. It saves the user information into a mongoDB cloud database and one can subtract an amount, reset it, or create a new total spending amount. I built a web version as well so that you can see how it functions if you don't have android or don't want to download it onto your phone. The front-end and back-end are hosted on Heroku.

You can test it out with username: 'Test' and password: 'Test' for yourself to see how it functions, or create your own account!
