This project focuses on front-end design to reimagine a pre-existing site that had been built on Squarespace (with permission from the church). It is built on Bootstrap to take advantage of their grid system and their carousel, but most other aspects of the design have been built on CSS.

It has also been built to include a user/admin login, which will redirect the homepage to a member’s blog page. It utilizes RESTful routing API where users and admins can update and delete the posts.

You can log in with username: "Joe Smith" and password: "joe", or you can log in as the admin with username: "Admin Ben" and password: "admin".
